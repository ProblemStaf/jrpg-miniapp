# 🔮 Поиски Тайн - Telegram Mini App RPG

Браузерная RPG-игра для Telegram с полноценным бэкендом, базой данных и социальными функциями.

## ✨ Возможности

### Геймплей
- 🗺️ Исследование карты с перемещением персонажа
- ⚔️ Пошаговая боевая система
- 🎒 Система инвентаря с предметами
- 🔨 Крафт вещей из ресурсов
- 📈 Прокачка через опыт и уровни
- 🎁 Ежедневные награды
- 🏆 Лидерборд игроков

### Интеграция с Telegram
- 👥 Реферальная система с бонусами
- 🌐 Telegram WebApp с нативной темизацией
- 📱 Адаптация под мобильные устройства
- 💳 Подготовка к платежам Stars

### Технические особенности
- 🗄️ SQLite база данных для хранения прогресса
- 🚀 Асинхронный Telegram бот
- 🐳 Docker для удобного деплоя
- 🎨 Темная/светлая тема от Telegram

## 🚀 Быстрый старт

### 1. Клонирование
```bash
git clone <repository-url>
cd jrpg-miniapp
```

### 2. Настройка
```bash
cp .env.example .env
# Отредактируйте .env, добавив ваш токен бота
```

### 3. Установка зависимостей
```bash
pip install -r requirements.txt
```

### 4. Запуск
```bash
python server.py
```

Сервер запустится на `http://localhost:5000`

## 🐳 Docker запуск

```bash
docker build -t jrpg-miniapp .
docker run -p 5000:5000 --env-file .env jrpg-miniapp
```

## 🎮 Игровые механики

### Боевая система
- **Атака**: Нанесение урона врагу
- **Лечение**: Использование зелий
- **Побег**: Выход из боя

### Крафт
Рецепты:
- 🧪 Зелье маны (Трава x2, Вода x1)
- ⚔️ Стальной меч (Железо x3, Дерево x2)
- 🛡️ Броня (Кожа x5, Железо x2)

### Прокачка
- Опыт получается за бои и ежедневные награды
- Каждый уровень увеличивает максимальное HP на 20
- Награда за уровень: Зелье опыта

## 🔧 Переменные окружения

| Переменная | Описание | Пример |

| `TELEGRAM_BOT_TOKEN` | Токен бота от @BotFather | `123456:ABC-DEF...` |

| `PORT` | Порт сервера | `5000` |
| `DEBUG` | Режим отладки | `True/False` |
