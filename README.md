# Dungeon Crawler: Roguelike

Классический roguelike в браузере с процедурной генерацией подземелий.

## 🎮 Особенности

- **Процедурная генерация** — каждый этаж уникален
- **Пошаговая боевая система** — продумывайте ходы
- **Инвентарь и экипировка** — оружие, броня, зелья
- **Прокачка персонажа** — уровни, опыт, улучшения
- **Таблица лидеров** — сохраняйте лучшие результаты
- **Адаптивный дизайн** — игра на ПК и мобильных

## 🚀 Быстрый старт

### Через Python

```bash
# Установка зависимостей
pip install -r requirements.txt

# Запуск сервера
python server.py
```

Откройте http://localhost:5000 в браузере.

### Через Docker

```bash
# Сборка образа
docker build -t dungeon-crawler .

# Запуск контейнера
docker run -p 5000:5000 dungeon-crawler
```

## 🎯 Управление

| Клавиша | Действие |
|---------|----------|
| WASD / Стрелки | Перемещение |
| SPACE | Пропустить ход |
| E | Подобрать предмет |
| Клик по предмету | Использовать/Экипировать |

## 📦 Предметы

- 🍷 **Зелье лечения** — восстанавливает 25 HP
- 🗡️ **Деревянный меч** — +5 урона
- ⚔️ **Железный меч** — +10 урона
- 🛡️ **Броня** — снижает входящий урон
- 💰 **Золото** — валюта (будет магазин)
- 📜 **Свиток огня** — 30 урона всем врагам

## 🔧 Конфигурация

Скопируйте `.env.example` в `.env` и настройте:

```env
PORT=5000        # Порт сервера
DEBUG=False      # Режим отладки
```

## 📱 Мобильная версия

Игра адаптирована для мобильных устройств:
- Сенсорное управление (через экранные кнопки в будущем)
- Адаптивный интерфейс
- Оптимизированный canvas

## 🛠 Расширение

Для добавления новых функций:

1. **Новые предметы** — добавьте в `ITEMS` в `game.js`
2. **Новые враги** — расширьте массив `ENEMIES`
3. **Сохранение в БД** — подключите SQLite в `server.py`
4. **Мультиплеер** — используйте WebSocket для синхронизации
